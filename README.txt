Plug «WpCalc Modal Form» is intended to create a variety of forms, including modal feedback window.

== Installation ==

* Download the folder wpcalc-modal-form in the plugins directory `/wp-content/plugins/`
* Activate «Wpcalc Modal Form» in the “Plugins” section.
* Go to plugin setup section and install the necessary options.


To insert a contact form on the page or on the site entry use the shortcode [wpcalcmf].
